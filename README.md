### Project Features
1. Automatically reads incoming client emails and extracts sender details and received date
2. AI analyzes the email and generates a clear task summary
3. Smart assignment system selects the best expert (Web / AI / Software) based on the task
4. Automatically sets “Assigned On” date and a 10-day deadline
5. Stores all structured data in a Notion database (Client Name, Client Email, Task, Assigned On Date, Assigned To, Deadline)
6. Saves the full original email inside the Database page for reference
7. Sends an automatic reply (with subject and message) to the client confirming task receipt
